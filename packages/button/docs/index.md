# button

## Attribute
|参数|说明|类型|可选值|默认值|
|-|-|-|-|-|
|type|类型|string|default / primary / 2d|default|
|size|尺寸|string|large / medium / small / mini|large|

## Events
|事件名称|说明|回调参数|
|-|-|-|
|click|点击按钮触发|(event: Event)|
